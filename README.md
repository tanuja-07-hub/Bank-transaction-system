# Bank Transaction System

##  Description
This is a Java-based console application that simulates a banking system. It allows users to perform operations like deposit, withdrawal, payment, and balance checking with secure PIN authentication.

##  Features
- PIN verification (3 attempts)
- Deposit money
- Withdraw money (multiples of 100)
- Make payments
- Check account balance
- Exit option

##  Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner class for input handling

##  How to Run
1. Compile the program:
   javac BankTra.java

2. Run the program:
   java BankTra
